# Reading assignment 

## Article: [Beyond the code: Preparing software engineers for the AI era](https://news,ontariotechu.ca/archives/2026/01/beyond-the-code-preparing-software-engineers-for-the-ai-era.php)

## What I found interesting

This article discusses the role of software engineers in the era of Artificial Intelligence. It highlights how students now should use AI to elevate their capabilities, and focus on being able to correct AI generated code when it makes a mistake. It also highlights risks regarding why AI generated code isnt the greatest, such as concerns about security, scalability, and reliability. 

I think this article is a good read because we are currently CS students entering the AI era, and it is important to know how to adapt to this new landscape
